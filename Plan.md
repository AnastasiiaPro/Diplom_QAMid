# План по проверке и автоматизации приложения «Мобильный хоспис».

## Описание приложения

Приложение даёт функционал по работе с новостями хосписа и включает в себя:

- информацию о новостях и функционал для работы с ними;
- цитаты;
- информацию о приложении.

## Исходные данные

- Архив с приложением;
- Учётные данные для авторизации в приложении: логин: login2 пароль: password2.

## 2 этап. Типы тестирования.

В процессе тестирования мобильного приложения применяются различные типы тестирования, каждый из которых проверяет определённые аспекты работы приложения. Вот основные типы тестирования и то, что именно будет проверяться в каждом из них:

1. Функциональное тестирование:

- Корректная работа всех функций приложения (авторизация, выход, переход между экранами, создание, редактирование, удаление новостей и т.д.).
- Соответствие функциональности заявленным требованиям.
- Отсутствие ошибок при взаимодействии с приложением (например, ввод данных, фильтрация новостей).

2. Тестирование юзабилити (Usability Testing):

- Ожидаемое поведение при взаимодействии (переходы по ссылкам, действия при нажатии кнопок).

3. Тестирование пользовательского интерфейса (UI Testing):

- Работа всех элементов интерфейса (кнопки, выпадающие меню, поля ввода).
